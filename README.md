# 🧩 How JavaScript Works

A comprehensive guide to understanding **how JavaScript works under the hood**.

Instead of memorizing concepts, this repository explains the JavaScript runtime step by step with practical examples, tables, code snippets, and visual explanations.

---

## 📚 Topics Covered

- JavaScript Engine
- V8, SpiderMonkey & JavaScriptCore
- Tokenization
- Parsing & AST
- JIT Compilation
- Execution Context
- Global Execution Context
- Function Execution Context
- Memory Creation Phase
- Execution Phase
- Call Stack
- Hoisting
- Temporal Dead Zone (TDZ)
- Scope
- Scope Chain
- Lexical Environment
- Closures
- Closure Use Cases
